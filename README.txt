BALANCEREWARD WAITLIST + SUPABASE

Archivos principales:
- index.html
- demo.html
- funcionalidad.html
- setup.html
- styles.css
- app.js
- supabase-config.js

Qué hace:
- La home incluye formulario de waitlist.
- El formulario inserta registros en Supabase.
- setup.html explica cómo crear la tabla y configurar las claves.

IMPORTANTE:
- Debes rellenar supabase-config.js con tu URL y tu anon key pública.
- Debes crear la tabla waitlist_signups en Supabase.
- Esta versión no crea cuentas con contraseña; sirve para waitlist / acceso anticipado.
